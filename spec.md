# 📘 Project Specification - Todo App

## 🎯 Objective
Create a modern task management web application.

## ⚙️ Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks
- Save tasks using localStorage
- Toggle dark/light theme

## 🎨 UI Requirements
- Clean modern design
- Responsive layout
- Smooth animations

## 🛠️ Technologies
- HTML, CSS, JavaScript
